##Se carregar na tecla 'g' durante o New Game Screen, ativa o "God Mode" em que o heroi não morre##

Se for a primeira vez que o jogador entra no jogo a unica opção é "New Game" mas se o o jogador passar do primeiro nivel 
passa a apresentar a a opção "Continue" que o devolve á primeira sala do nivel em que estava.

Em caso de Boss Room o Boss deve ser o ultimo a morrer para conseguir a Skeleton Key, caso contrario o ultimo inimigo a morrer dropa uma regular key.
